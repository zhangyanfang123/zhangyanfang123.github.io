Input Date 对象

Input Date 对象是 HTML5 中的新对象。
Input Date 对象表示 HTML < input type=”datetime”> 元素.注释：Internet Explorer 或 Firefox 不支持 < input type=”date”> 元素。

1.日期时间( < input type=”datetime-local”/>)

既显示日期组件，又显示时间组件。
